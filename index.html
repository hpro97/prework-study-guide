HTML

  • "!DOCTYPE html" tells browser page is written in html format
  • "html lang="eng" tells browser that the language is in English
  • "head" element contains things such as meta data and important data about the page
  • "body" element defines the body of the document, which contains the content that is visible to the user.
  • "p" element represents a paragraph or block of text.
  • "ul", "ol", and "li" elements represent unordered lists, ordered lists, and list items.
  • "img" element contains information about images that are displayed on the webpage.
  • "src" attribute is the most important because it defines the location of the image
  • "br" element creates a line of empty space, or a line break, between two blocks of content.
  • "a" element (which stands for anchor) creates links to the same webpage or other webpages.
  • "href" attribute points to the URL for the link
  • "section" elements are semantic elements that define a section in a document that contains thematic content.

CSS

  • CSS rule consists of selectro and declarations
  • semi colons seperate, colons before defining value
  • to select a unique component on a page, you can use the # symbol followed by the id of the element.
  • use . to select class
  • in class selection use , to seperate class selector
  • a CSS comment starts and ends with /* */
  • small list of commands:
    background-color: red;
    background-image: url("cat.gif");
    margin-top/bottom/left/right: 5px;
    background-image: url("cat.gif");
    border-radius: 20px;
    font-family: sans-serif;
    color: red;
    padding: 5px;
    img{height/width: 100px;}
    padding: 5px;
  • a margin indicates how much space we want around the outside of an element.
  • a padding indicates how much space we want around the content inside an element.

JavaScript

  • var defines the variable
  • Control flow is the order in which a computer executes code in a script.
  • in between speechmarks is the string (or whatever thing or list of things known as a string is assigned to a variable)
  • console.log method is used to output a message to the web console by adding an argument
  • whatever is inside parentheses is what will be output, if its a variable it will fetch the string associated
  • just like css a js file is added to HTLM, still by a link element but in the body element as opposed to the head like with css
  • When the computer encounters an if statement in a JavaScript file, it will check to see if the condition that we provide is truthy before running curly brackets command, otherwise ignore
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • An array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
  • square brackets after variable can be used to grab single pieces of data from an array
